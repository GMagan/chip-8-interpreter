Para utilizar você deve seguir as instruções de compilação presentes no outro README

Rode em seu terminal:
./c8interpreter.exe <arquivo_ROM>

Configurações Internas:
Taxa de atualização (frames): 60 Hz
Frequência da CPU: 500 instruções por segundo
Escala do display: 10 pixels por unidade

Esses valores podem ser ajustados diretamente nas variáveis globais do arquivo main.cpp.

Controles do Teclado CHIP-8

O mapeamento segue o padrão:

2	      2
